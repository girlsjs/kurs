Pętle i tablice

Pętle

W dzisiejszym ćwiczeniu poznamy pętle.

Pętle, podobnie jak instrukcja warunkowa "if", należą do podstawowych konstrukcji w językach programowania.

Pętle pozwalają na wykonanie danej czynności kilka razy (bez potrzeby kopiowania i wklejania kodu)

Ale...

Żeby nasze ćwiczenie miało sens, najpierw musimy się dowiedzieć co nieco o tablicach.

Tablice

Tablice to po prostu zmienne, które mogą przechowywać wiele wartości.

Zmienna

Jak pamiętamy z naszego warsztatu, zmienna to taki schowek, do którego możemy włożyć jakąś wartość.

Tablica

Więc tablica to taki zbiór schowków, z numerkami. Do każdego z nich też można włożyć jakąś wartość.

Po co to?

Żeby w ładny sposób przechowywać wiele podobnych danych

Przykład: Jeżeli mamy 10 użytkowników na stronie, to znacznie łatwiej przechować ich imiona w jednej tablicy niż w 10 oddzielnych zmiennych.

Jak to wygląda

Przypomnijmy sobie najpierw jak definiujemy zmienne

var name = "Kasia";

A tak wygląda tablica

var names = ["Kasia", "Basia", "Asia"];

Lub tak

var names = [
  "Kasia",
  "Basia",
  "Asia"
];

Jak odczytać wartość

Przypomnijmy sobie jak odczytać wartość zmiennej

console.log(name);

Wykorzystajmy numerki w tablicy

console.log(names[1]);

Uwaga! Dziwne numerowanie

W tablicach zaczynamy numerować od wartości 0

var names = [
  "Kasia", // To jest wartość o numerze 0
  "Basia", // To jest wartość o numerze 1
  "Asia"   // To jest wartość o numerze 2
];

Pętle

Jest kilka rodzajów pętli.
Ale dziś skupimy się na jednej - pętli for.

Jest to najczęściej używana pętla.

Anatomia pętli for

for (var i=0; i<10; i++) {...}
  • słowo kluczowe for
  • tworzenie zmiennej
    (jaka zmienna ma być użyta do kolejnych cykli pętli)
  • warunek trwania pętli
    (pętlę kiedyś trzeba skończyć)
  • zwiększanie zmiennej
    (jaka ma być wartość zmiennej w kolejnych cyklach)

Porównajcie do instrukcji warunkowej if

if

if (warunek) {
  // tutaj nasz kod
}

for

for (var i=0; i<10; i++) {
  // tutaj nasz kod
}

Spróbujcie wpisać w konsolę następujące

for (var i=0; i<5; i++) { console.log(i) }

Ćwiczenie

Ćwiczenie

Wykorzystując zdobytą dziś wiedzę:

  • Utwórzcie tablicę z 6 imionami
  • Wypiszcie, używając pętli, wszystkie imiona

Prawdopodobnie wyszło Wam coś takiego

var names = ["Ala", "Ela", "Ola", "Ula", "Ila", "Yla"];

for (var i=0; i<6; i++) {
  console.log(names[i]);
}

Problem z takim rozwiązaniem

Nasze rozwiązanie działa tylko dla tablicy z 6 imionami.

Co jeżeli będzie ich mniej lub więcej?

"length" na ratunek!

Każda tablica ma właściwość length

length zawiera zawsze aktualną ilość elementów w tablicy

names.length // wypisze ilość elementów

Wracamy do ćwiczenia

Rozwiążcie poprzednie zadanie tak, żeby działało dla dowolnej liczby imion w tablicy names

Zgadza się?

var names = ["Ala", "Ela", "Ola", "Ula", "Ila", "Yla"];

for (var i=0; i<names.length; i++) {
  console.log(names[i]);
}

Czas na coś innego

Zanim przejdziemy do ćwiczenia końcowego, nauczymy się jeszcze jednej rzeczy.

Modyfikowanie drzewa DOM

Czym jest drzewo DOM?

Document Oriented Model - jest to dokument napisany w HTMLu i wyświetlony w przeglądarce

Czyli dokładnie to, co widzicie w narzędziach programisty w zakładce Elementy

Czas odpalić edytor

Przejdźcie do swojego ulubionego edytora i otwórzcie plik excercise1.html

excercise1.html

<body>
  <section id="section1">
    <h2>Nagłówek 1</h2>
    <p>Lorem ipsum dolor sit amet...</p>
    ...
  </section>

  <script>
    // tutaj miejsce na Wasz skrypt ;-)
  </script>
</body>

Tworzenie nowego elemenu

Aby utworzyć nowy element, należy użyć metody document.createElement()

Jako argument podajemy co to ma być za element (p, div, h1...)

var p = document.createElement('p');

W ten sposób utworzyliśmy pusty akapit

A jak umieścić w nim jakiś tekst?

Poprzez ustawienie wartości innerText

var p = document.createElement('p');
p.innerText = 'mój napis';

Tak utworzony element jest wirtualny

Istnieje w pamięci przeglądarki, ale nie jest nigdzie umieszczony w drzewie DOM

Jak go doczepić?

Gdzie?

Najpierw musimy znaleźć element do którego chcemy go doczepić

Jak znaleźć element, kto pamięta?

A tak

document.querySelector()

Jeżeli chcemy znaleźć element o identyfikatorze (id) section1 ...

document.querySelector('#section1');

Dodawanie elementu

Do tego służy metoda appendChild

Używamy jej w ten sposób: nadrzędnyElement.appendChild(nowyElement)

Jak to będzie wyglądało u nas?

document.querySelector('#section1').appendChild(p)
